全链路创作工具箱

从数据采集到灵感生成，一次铺设，长期复用

📡

多平台同步

知乎收藏夹、B站视频（字幕/AI摘要）、小宇宙热门 — 增量同步，断点续传，按账号隔离。

🧠

向量知识库

ChromaDB 中文语义搜索，可筛选平台/作者/收藏夹。自动去重、增量构建，不重复计算。

🌲

知识树

从主题簇自动生成层级知识树。手动 Override 规则，选中文本即可匹配对应节点。

📊

主题聚类

KMeans 自动发现 15 个主题簇，LLM 为每簇生成摘要、核心观点、反方视角和选题建议。

🃏

素材卡片

LLM 从每篇内容抽取关键论据、金句、案例。两种质量等级：知乎全量卡，B站轻量卡。

✍️

AI 写作助手

输入问题 → 检索知识库 + 卡片 + 主题 → 生成结构化素材包 → 可选初稿。支持改写建议。

🔍

质量检查

逐段相似度分析，标注高相似段落（>0.45），自动生成改写建议和来源依赖分布。

🌐

Web 仪表盘

Streamlit 搜索面板、知识图谱与主题网络可视化、词云与机会分、AI 写作与质量检查。

🔌

浏览器扩展

Chrome MV3，右键选中文本 → 本地 API 分析 → 侧边栏：节匹配、相似素材、写作风险提醒。

👤

多账号

--account 参数隔离知乎多账号。Cookie、状态、知识库、导出各自独立，互不干扰。

📦

JSONL 导出

source_items（去重+质量分）→ chunks（RAG 分块）→ corpus（style/evidence 分层）。AI 即用。

🔒

隐私优先

所有数据本地存储。Cookie 自行管理，API Key 仅用于 DeepSeek 调用。来源全程可追溯。

5 分钟上手

Python 3.10+，一条命令安装

# 安装
$ git clone git@github.com:paimmme/zhihu2obsidian.git
$ cd zhihu2obsidian/cli
$ python -m venv .venv && source .venv/bin/activate
$ pip install -e ".[all]"

# 初始化 + 导入 Cookie（从浏览器 DevTools 复制 z_c0 和 d_c0）
$ zhihu2obsidian config init
$ zhihu2obsidian auth login

# 同步 → 构建知识库 → 启动
$ zhihu2obsidian sync
$ zhihu2obsidian knowledge topics build
$ zhihu2obsidian knowledge tree build
$ zhihu2obsidian serve
🚀 本地 API: http://127.0.0.1:8765
# 另一个终端打开 Streamlit 仪表盘
$ streamlit run web/app.py

1 日常同步

# 增量同步所有平台
$ zhihu2obsidian sync
$ zhihu2obsidian bilibili sync

# 月度一键全量
$ zhihu2obsidian monthly

2 搜索与写作

# 语义搜索（支持平台/作者/收藏夹筛选）
$ zhihu2obsidian search "AI 编程" --platform bilibili

# AI 写作（需 DeepSeek Key）
$ zhihu2obsidian write "古法编程" --package --draft
