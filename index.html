功能

从数据采集到 AI 写作，全链路打通的个人知识工作台

📡

多平台同步

知乎收藏夹、B站视频（字幕/AI摘要）、小宇宙热门播客 — 一键增量同步。

🧠

向量知识库

ChromaDB 中文语义搜索，按平台/作者/收藏夹筛选，自动去重与增量构建。

🌲

知识树

从主题簇自动生成层级知识树，支持手动调整、文本节点匹配、覆盖规则。

📊

主题聚类

KMeans 自动发现素材中的 15 个主题簇，LLM 生成摘要、观点、选题建议。

🃏

素材卡片

LLM 从每篇文章提取核心观点、论据、案例、金句、反方视角和写作角度。

✍️

AI 写作

输入问题 → 自动检索知识库+卡片+主题 → 生成结构化素材包 → 可选初稿。

🔍

质量检查

逐段相似度分析，标注高相似段落，自动生成改写建议。

🔌

浏览器扩展

Chrome MV3 扩展，右键选中文本 → 本地 API 分析 → 侧边栏显示知识树匹配/素材/风险。

👤

多账号

支持多知乎账号隔离，--account 参数区分同步、知识库、导出，数据互不干扰。

📦

JSONL 导出

三层格式：source_items（去重+质量分）、chunks（分块RAG单元）、corpus（style/evidence）。

🌐

Web 仪表盘

Streamlit 5 面板：搜索、知识图谱、词云、主题详情、AI 写作。

🔒

隐私优先

所有数据存储在本地。Cookie 本地管理，API Key 本地配置，来源全程可追溯。

快速开始

5 分钟从零到导出第一篇收藏

1

安装

Python 3.10+，pip install

# 克隆项目
$ git clone git@github.com:paimmme/zhihu2obsidian.git
$ cd zhihu2obsidian/cli
$ python -m venv .venv && source .venv/bin/activate
$ pip install -e ".[all]"

# 初始化 + 导入 Cookie（从浏览器 DevTools 复制 z_c0 和 d_c0）
$ zhihu2obsidian config init
$ zhihu2obsidian auth login
2

同步收藏夹

首次同步建议加 --limit 3 测试

$ zhihu2obsidian list
📂 共 2 个收藏夹
  1. 我的收藏 (994375688) — 12 篇
  2. tech (1003429158) — 9 篇

$ zhihu2obsidian sync
✅ 同步完成: 21 篇 / 35 张图片
3

构建知识库 + 启动仪表盘

$ zhihu2obsidian knowledge build
🧠 向量化... 存储 398 个向量
🔗 知识图谱...
☁️ 词云...

$ streamlit run web/app.py
You can now view your Streamlit app at:
http://localhost:8501

日常使用

一次 sync，随时检索

# 搜索素材
$ zhihu2obsidian search "AI 编程工具" --platform zhihu
🔎 搜索结果 (去重):
  #1 AI编程Agent工具对比与选型实践 (score: 0.630)
  #2 RAG 系统设计与实现 (score: 0.487)

# 素材卡片抽取（一次抽取，后续增量）
$ zhihu2obsidian knowledge cards build
📄 已抽取 22 张素材卡片

# 主题聚类 + 知识树
$ zhihu2obsidian knowledge topics build
📂 共 15 个主题簇
$ zhihu2obsidian knowledge tree build
🌲 知识树已构建，15 节点

# 启动本地 API + 浏览器扩展
$ zhihu2obsidian serve
🚀 本地 API: http://127.0.0.1:8765

# AI 写作（需 DeepSeek API Key）
$ zhihu2obsidian write "古法编程值不值得坚持" --package --draft --check
✍️ 生成完成。相似度检查: ✅ 原创度良好

# 月度一键全量
$ zhihu2obsidian monthly
# 多账号
$ zhihu2obsidian sync --account personal
$ zhihu2obsidian export --no-cards --account personal
